## 🎮 Game Modes

- **Hot Takes**: Viral voting on spicy opinions.
- **Emoji IQ**: Decode the hidden meaning of emojis.
- **Vibe Quiz**: Discover your personality archetype.
- **Vibe or Die**: High-stakes reaction battle.
- **Quiz Arena**: Real-time multiplayer trivia.
- **Reaction Arena**: Test your reflexes against friends.
- **Geography Guesser**: Identify flags from around the globe.
- **And many more...**

## 📚 Features
- **Zero Friction**: No account creation or login needed.
- **Expert Guides**: Learn the psychology and strategies behind your favorite games at `/guides`.
- **Global Multiplayer**: Create private rooms and challenge friends in seconds.
- **Built for 2026**: Modern tech stack with a focus on speed, aesthetics, and user privacy.

## 🛠️ Tech Stack
- **Framework**: [Next.js](https://nextjs.org/) (App Router)
- **Styling**: Vanilla CSS with modern aesthetics (Glassmorphism, Neon glow)
- **Icons**: [Lucide React](https://lucide.dev/)
- **Real-time**: Google Firebase (for multiplayer features)
- **Deployment**: Custom domain (vibemenow.uk)

## 🛠️ Development
### Getting Started
First, clone the repository and install dependencies:
```bash
npm install
```

Then, run the development server:
```bash
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.
